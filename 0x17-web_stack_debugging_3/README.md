    <p style="box-sizing: border-box; margin: 0px 0px 10px;">When debugging, sometimes logs are not enough. Either because the software is breaking in a way that was not expected and the error is not being logged, or because logs are not providing enough information. In this case, you will need to go down the stack, the good news is that this is something Holberton students can do :)</p>
    <p style="box-sizing: border-box; margin: 0px 0px 10px;">Wordpress is a very popular tool, it allows you to run blogs, portfolios, e-commerce and company websites&hellip; It&nbsp;<a href="https://intranet.hbtn.io/rltoken/Ah9_LmUi191dqxT-Zx7uhg" style="box-sizing: border-box; background-color: transparent; color: rgb(224, 0, 60); text-decoration: none;" target="_blank" title="actually powers 26% of the web">actually powers 26% of the web</a>, so there is a fair chance that you will end up working with it at some point in your career.</p>
    <p style="box-sizing: border-box; margin: 0px 0px 10px;">Wordpress is usually run on LAMP (Linux, Apache, MySQL, and PHP), which is a very widely used set of tools.</p>
    <p style="box-sizing: border-box; margin: 0px 0px 10px;">The web stack you are debugging today is a Wordpress website running on a LAMP stack.</p>
